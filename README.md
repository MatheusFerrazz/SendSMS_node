## O Projeto

Código que implementa o envio de SMS para celulares por meio de uma API privada chamada TeleSign. Esse tipo de serviço é útil para altenticação via mensagem de Texto por exemplo. 

## As Tecnologias

- Node.js
- API TeleSign
- SDK TeleSign
